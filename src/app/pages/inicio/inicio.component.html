Descripción del Proyecto

Este proyecto tiene como objetivo ofrecer un sistema de registro y login para los usuarios, complementado con un sistema de chat dividido en grupos de trabajo. Los usuarios podrán enviar mensajes que se almacenarán diariamente, accesibles a través de un calendario, permitiendo la inclusión de códigos y archivos adjuntos en los mensajes.

Resumen del Backend y SignalR

El backend, construido con ASP.NET Core, está diseñado para ser eficiente y escalable, utilizando SignalR para la comunicación en tiempo real. Esto permite al servidor enviar mensajes instantáneamente a los clientes, facilitando una experiencia de chat fluida y dinámica.

Clases Backend

Las clases fundamentales del backend incluyen Connection, para gestionar conexiones de usuario; Message, para definir la estructura de los mensajes; y ResultadoHash, para manejar los resultados de las operaciones de hashing.

SignalR en Nuestro Sistema de Chat

SignalR es una biblioteca avanzada para ASP.NET Core que posibilita la comunicación en tiempo real entre el servidor y los clientes web. Es la piedra angular de nuestro sistema de chat, permitiendo interacciones dinámicas y en vivo sin la necesidad de recargar la página.

Características Clave Implementadas con SignalR

1. Salas de Chat Dinámicas

Utilizamos SignalR para crear salas de chat donde los administradores pueden asignar empleados específicos. Estas salas facilitan la colaboración enfocada y permiten reuniones virtuales dentro de grupos de trabajo.

2. Gestión del Historial de Mensajes

A través de la integración de SignalR con nuestra base de datos, proporcionamos a los usuarios acceso al historial completo de mensajes en cualquier sala, garantizando que la información importante nunca se pierda.

3. Compartir Archivos

Nuestro chat permite a los usuarios compartir archivos directamente en la conversación. SignalR juega un papel crucial aquí, notificando a todos los miembros de la sala cuando un nuevo archivo está disponible.

4. Filtrado de Lenguaje

Para mantener un entorno profesional, implementamos un filtro de lenguaje que examina y bloquea el contenido inapropiado en tiempo real, asegurando que las conversaciones permanezcan respetuosas y productivas.

Seguridad en Nuestro Sistema de Chat

La seguridad es un aspecto crucial de nuestro sistema de chat, asegurando que solo los usuarios autorizados puedan acceder y que la información se transmita de manera segura.

Autenticación JWT

Implementamos la autenticación basada en tokens JWT (JSON Web Tokens) para gestionar de forma segura las sesiones de usuario. Esto permite a los usuarios autenticarse una vez y luego realizar solicitudes al servidor sin tener que enviar sus credenciales en cada solicitud.

Gestión de CORS

Para permitir que nuestra aplicación Angular interactúe con el backend de ASP.NET Core, configuramos la política CORS (Cross-Origin Resource Sharing) para aceptar solicitudes de dominios específicos, asegurando que solo las solicitudes de origen confiable puedan acceder a los recursos del servidor.

Servicios Adicionales

Además de la autenticación y la gestión de CORS, nuestro backend incorpora varios servicios para enriquecer la funcionalidad del chat, como el servicio de historial para almacenar y recuperar mensajes anteriores y el servicio de limpieza para manejar la eliminación de datos antiguos o innecesarios.
